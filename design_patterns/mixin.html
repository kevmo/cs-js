Mixins = classes that offer functionality that can easily be inherited by a sub-class or groups thereof for the purpose of function reuse. "Subclassing" = inheriting properties for a new object from a base or 'superclass' object. Class B can extend Class A --> Class B is the subclass, Class A is the superclass. Backbone pattern: Backbone calls _.extend(Model.prototype, Events) on the Backbone primitives prototype objects, incl. Models, Views, Collection, Router, and History.
